INTRODUCTION

What is Aeris?

💬

AI Customer Support Chatbot

Aeris helps businesses answer customer questions instantly by connecting a chatbot to company documents, FAQs, policies, and website content.

⚠️

Problem We Solve

Many businesses still depend on manual support for repeated questions, causing slow response times, inconsistent answers, and higher support workload.

MOTIVATION

Why Aeris Matters

🤖

Reduce Repetitive Work

Automates common questions so support teams can focus on complex cases.

✅

Consistent Answers

Uses approved company knowledge to keep responses reliable and aligned.

🕒

24/7 Availability

Provides support outside working hours without needing extra staff.

🔍

Knowledge Gap Detection

Helps identify missing information when the chatbot cannot answer confidently.

TARGET MARKETS

Built for Any Industry

One platform, different knowledge bases. Aeris can support multiple industries by adapting to each company's own data.

🏥

Healthcare & Pharmacy

Patients and caregivers can get instant answers about appointments, medication guidance, clinic policies, and general support FAQs.

🎓

Education

Students and staff can get help with LMS navigation, course information, school services, and IT support questions.

🛒

E-Commerce

Customers can receive real-time help with product information, order status, returns, refunds, and post-purchase support.

USE CASE

How Users Interact With Aeris

  • System Admin

    System Admin manages access and backend settings.

  • User Admin

    User Admin uploads and manages company knowledge sources.

  • User

    User submits support questions through the chat interface.

  • Admin Review

    Admin reviews unanswered questions and improves the knowledge base.

IMPLEMENTATION / TOOLS

Project Tools & Technologies

Planned tools from the PRD.

🖥️
Front-end

Streamlit / React

Development Tools Visual Studio Code
Language Python / JavaScript / TypeScript
Framework Streamlit / React

Streamlit is used to build interactive web apps with Python. React supports reusable JavaScript components for building user interfaces efficiently.

☁️
Back-end

LangChain / LangGraph

Development Tools Visual Studio Code
Language Python

The chatbot backend will use LangChain and LangGraph to build and manage agents, decide which tools to use, stream responses, and integrate chat models and embedding models for LLM workflows.

🗄️
Database

MySQL / ChromaDB

Development Tools MySQL Workbench / ChromaDB
Language MySQL / Python

MySQL stores user admin, platform management, and uploaded document records. ChromaDB stores embeddings with metadata and enables vector search to retrieve relevant knowledge-base chunks.

🛠️
Development Tools

VS Code / Android Studio / Notepad++ / MySQL Workbench

These tools support coding, editing, debugging, and database management throughout the project development process.
